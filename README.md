# Assignment 02 – Automated Testing Framework
This repository contains the automated test framework developed for CSE2522 – Assignment 02 using:
- Selenium WebDriver
- NUnit Testing Framework
- C# (.NET)
- Page Object Model (POM) Design Pattern

The automation is based on the manual test cases provided in the assignment Excel sheet and is implemented using industry-standard test automation practices.
